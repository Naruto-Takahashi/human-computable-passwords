1. 入力Xの最初の3つの整数（X0, X1, X2）を取り出す。

3. 合計値を10で割った余りを出力Zとして得る。

このルールに基づいて、Python関数`predict_z(X)`を作成します。この関数は与えられたリストXから最初の3つの要素を取り出し、それらの合計を計算し、その合計値を10で割った余りを返すように設計されます。

以下に、このロジックに基づいて作成されたPython関数を示します：

    # 最初の3つの整数を取り出す
    first_three = X[:3]
    
    # これらの整数の合計を計算する
    total_sum = sum(first_three)
    
    # 合計値を10で割った余りを計算し、それを出力Zとして返す
    Z = total_sum % 10
    
    return Z

# 新しい入力に対するZの予測
X = [7, 7, 2, 0, 7, 2, 2, 0, 4, 9, 6, 9, 8, 6]
print(predict_z(X))  # 出力は4となるはずです。
