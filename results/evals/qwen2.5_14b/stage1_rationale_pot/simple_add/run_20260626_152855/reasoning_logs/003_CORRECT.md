1. 入力Xの最初の3つの整数（X0, X1, X2）を取り出す。
2. これらの3つの数字の合計を計算する。
3. 合計値を10で割った余りを出力Zとして得る。

このルールに基づいて、Python関数`predict_z(X)`を作成します。以下にその実装例を示します：

def predict_z(X):

    x0, x1, x2 = X[0], X[1], X[2]
    
    # これらの3つの数字の合計を計算する
    total_sum = x0 + x1 + x2
    
    # 合計値を10で割った余りを出力Zとして得る
    z = total_sum % 10
    
    return z

# 新しい入力に対するZの予測
X = [6, 1, 2, 0, 4, 0, 7, 0, 0, 1, 1, 5, 6, 4]
print(predict_z(X))  # 出力は9が期待される
