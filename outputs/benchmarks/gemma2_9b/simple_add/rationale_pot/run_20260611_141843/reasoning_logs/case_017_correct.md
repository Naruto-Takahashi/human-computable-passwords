1. 入力データの最初の3つの数字 (X0, X1, X2) を選択する。

これをPython関数として実装します。

  """
  入力データの最初の3つの数字を合計し、10で割った余りを返す関数です。

  Args:
    X: 14個の整数のリスト

  Returns:
    Z: 出力値 (0から9までの整数)
  """

  return z

* `sum_of_three % 10` は、合計を10で割った余りを計算するモジュール演算子を使用しています。この余りが出力値 Z となります。
* `return z` で計算された Z を返します。
