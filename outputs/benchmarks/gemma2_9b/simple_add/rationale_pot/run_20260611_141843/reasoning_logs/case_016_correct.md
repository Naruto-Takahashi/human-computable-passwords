def predict_z(X):
  """
  入力データの最初の3つの数字を合計し、10で割った余りをZとして出力する関数です。

  Args:
    X: 14個の整数のリスト

  Returns:
    Z: 0から9までの整数
  """

  z = sum_of_three % 10
  return z

* 合計を `% 10` で割ることで、10で割った余りを計算し、これを `z` に代入します。
* 最後に、`z` の値を返します。
