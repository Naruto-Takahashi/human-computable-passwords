## 隠れたルールと予測

観察データから、Zの値は入力の各要素の合計ではなく、**特定の数字の出現回数に基づいて決定されている**と考えられます。

具体的には、以下の手順でZを計算するルールが考えられます。

1. 入力リスト内の **偶数の出現回数** をカウントする。
2. 算術演算を行わずに、そのカウント値をそのまま Z の値とする。

このルールに基づくと、提供された入力データの各サンプルに対して、Zの値は入力リスト内の偶数の出現回数と一致しています。


## Python 関数定義

```python
def predict_z(X):
  """
  入力リスト X から Z を予測する関数

  Args:
    X: 14 個の整数のリスト

  Returns:
    Z: 入力リスト内の偶数の出現回数
  """
  count = 0
  for num in X:
    if num % 2 == 0:
      count += 1
  return count
```
