## 隠れたルールと予測

観察データから、Zの値は入力の各数字の合計に依存していると考えられます。しかし、単純な合計だけでは説明できない部分も存在します。

より深く分析すると、以下のパターンが見受けられます。

1. **奇数番目の数字の合計** と **偶数番目の数字の合計** をそれぞれ計算し、その差を算出する。
2. この差に **特定の定数** を加算する。
3. 最後に、その結果を **10で割った余り** が Z の値となる。

具体的な定数はデータから推測することが困難です。そのため、この関数の定義には定数を `k` と仮定し、ユーザーが適切な値を設定できるようにします。


## Python 関数

```python
def predict_z(X):
  """
  入力リスト X に対する Z を予測する関数

  Args:
    X: 14 個の整数のリスト

  Returns:
    Z: 0 から 9 の整数
  """
  odd_sum = sum(X[i] for i in range(1, len(X), 2))
  even_sum = sum(X[i] for i in range(0, len(X), 2))
  diff = odd_sum - even_sum
  z = (diff + k) % 10
  return z

```

**注意:** この関数は、定数 `k` を含んでいます。実際のデータに基づいて最適な `k` の値を決定する必要があります。
