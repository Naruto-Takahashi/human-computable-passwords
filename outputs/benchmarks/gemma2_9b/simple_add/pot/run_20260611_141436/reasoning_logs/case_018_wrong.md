## Python 関数 `predict_z(X)`

```python
def predict_z(X):
  total = sum(X)
  return total % 10
```



この関数は、入力リスト X の各数字の合計を計算し、その合計値を10で割った余りを返します。これが、与えられたデータセットにおける隠れたルールを表す関数です。
