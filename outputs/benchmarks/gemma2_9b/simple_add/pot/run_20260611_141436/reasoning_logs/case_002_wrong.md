## 隠れたルールと予測

観察データから、Zの値は入力の各数字の合計に依存していると考えられます。しかし、単純な合計だけでは説明できない複雑さを感じます。そこで、以下の手順で思考を進めます。

1. **各桁の数字をグループ化**: 入力データを4つのグループに分けてみます (X0-X3, X4-X7, X8-X11, X12-X13)。
2. **各グループの合計を求める**: 各グループの数字を合計します。
3. **グループ合計を足し合わせる**: 4つのグループ合計を全て足し合わせます。
4. **結果を10で割る余りをZとする**:  上記の合計を10で割った余りが、出力のZとなります。

このルールが全ての入力データに当てはまるか確認する必要があります。


## Python 関数 `predict_z(X)`

```python
def predict_z(X):
  group1 = sum(X[0:3])
  group2 = sum(X[4:7])
  group3 = sum(X[8:11])
  group4 = sum(X[12:])
  total = group1 + group2 + group3 + group4
  z = total % 10
  return z
```
