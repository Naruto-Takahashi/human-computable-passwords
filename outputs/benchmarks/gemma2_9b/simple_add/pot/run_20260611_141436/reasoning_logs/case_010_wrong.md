## 隠れたルールと予測

観察データから、Zの値は入力の各数字の和に依存していると考えられます。しかし、単純な合計では説明できない部分も存在します。

より深く分析すると、以下のパターンが見受けられます。

1. **奇数番目の数字の合計と偶数番目の数字の合計をそれぞれ計算する**
2. **それぞれの合計を足し合わせる**
3. **その和を 10 で割った余りを Z とする**

このルールに従うと、予測課題の入力に対してZは以下のようになります。

```
Input: [9, 8, 9, 4, 1, 3, 6, 7, 2, 0, 3, 1, 7, 3]
```

* 奇数番目の数字の合計: 9 + 9 + 4 + 1 + 6 + 2 + 3 + 7 + 3 = 45
* 偶数番目の数字の合計: 8 + 4 + 1 + 3 + 0 + 1 + 6 + 0 + 1 + 7 + 9 + 3 = 43

* 合計: 45 + 43 = 88
* 88 ÷ 10 の余り: 8


## Python 関数

```python
def predict_z(X):
  odd_sum = sum(X[i] for i in range(1, len(X), 2))
  even_sum = sum(X[i] for i in range(0, len(X), 2))
  total_sum = odd_sum + even_sum
  return total_sum % 10

```
